# Dating Website

A simple dating website built using Next.js and Firebase.

## Features
- User authentication
- Profile creation
- Gender filter
- Distance filter
- Interest filter
- Like and match system

## Tech Stack
- Next.js
- Firebase Authentication
- Firestore
- Tailwind CSS
- Vercel (deployment)

## How to Run Locally

1. Clone repository
2. Run npm install
3. Run npm run dev
4. Open http://localhost:3000

## Deployment

Deployed on Vercel:
